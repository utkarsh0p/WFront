# WovenWeave Agent Instructions

This project must be treated as a Vite + React frontend using Tailwind CSS and GSAP.

## Required Stack

- Vite
- React
- Tailwind CSS
- GSAP with `ScrollTrigger`

## Required Style Source

- Read `STYLES.md` before making any UI, layout, typography, animation, or component decision

## Visual Benchmark

- Use `https://hypefluency.com/` as the style and motion benchmark
- Preserve the premium, rounded, editorial, animation-heavy feel
- Adapt all content and visuals to a carpet business named `WovenWeave`

## Locked MVP Direction

- Pages: `Home`, `About`, `Collections`, `Gallery`, `Contact`
- Homepage: long-form and comprehensive
- Secondary pages: shorter supporting versions
- Language: English only
- Scope: frontend only
- Product focus: luxury carpets and rugs
- Brand tone: luxury editorial showroom with warm craft influence
- No backend
- No checkout
- No contact form logic
- No pricing required in the MVP
- Contact page can be static and visual

## Implementation Rules

- Keep all frontend work consistent with `STYLES.md`
- Prefer reusable React sections and Tailwind utility patterns
- Use GSAP `ScrollTrigger` for scroll-linked interactions
- Do not introduce a different design language unless explicitly requested
- Translate the reference style into textile, rug, craft, and interior storytelling
- Use reasonable assumptions and keep the MVP easy to customize later
